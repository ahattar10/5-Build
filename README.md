# Indie Film Collective Website

A multi-page website showcasing an independent film collective community dedicated to filmmakers, storytellers, and visual artists.

## Pages Included

- **Home (index.html)** - Welcome page introducing the Indie Film Collective
- **Film Screenings (screenings.html)** - Information about upcoming film screenings and events
- **Workshops (workshops.html)** - Details about hands-on learning opportunities
- **Collaborations (collaborations.html)** - Information about collaborative projects

## Features

- Mobile-first responsive design with CSS media queries
- Navigation menu on all pages with relative links
- Accessible images with descriptive alt text
- Clean, professional layout with dark theme
